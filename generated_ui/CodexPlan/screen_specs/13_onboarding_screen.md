# Screen Spec: Onboarding / First Create Guidance

## Screen ID

`SCR-ONBOARD-001`

## Purpose

This screen helps first-time users understand the product gently.

It should feel:
- warm
- hopeful
- simple
- non-technical

## UX Role

This screen supports:
- explaining the product value
- setting emotional expectations
- introducing the create flow
- preparing for permissions without sounding technical

## Primary Design Principle

Explain the feeling first, the mechanics second.

## Information Priority

1. emotional product promise
2. simple 3-step explanation
3. create-first CTA
4. optional permission hint

## Layout Structure

### Section 1: Welcome Hero
- warm visual/photo-led area
- title
- short value statement

### Section 2: 3-Step Story
- `写真を選ぶ`
- `思い出がまとまる`
- `メモとして残す`

### Section 3: Privacy / Permission Hint
- short calm language

### Section 4: CTA Area
- `はじめる`
- optional secondary `あとで`

## Components

- onboarding hero block
- step cards
- privacy hint card
- CTA footer
