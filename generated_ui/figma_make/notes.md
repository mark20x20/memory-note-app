# Figma Make 草案メモ

このフォルダは Figma Make で生成したUI草案の保管場所です。

## 重要
- ここのコードは **本番実装に直接使用しない**
- デザイン確認は Figma Design を正とする
- 実装本体は `app/` および `src/` に置く

## 構成
- `onboarding/` - オンボーディング草案
- `auth/` - ログイン・登録草案
- `home/` - ホーム画面草案
- `note_detail/` - ノート詳細草案
- `share_card/` - 共有カード草案
- `settings/` - 設定画面草案
- `screenshots/` - スクリーンショット

## 使い方
Figma Make のエクスポートをここに置き、
`src/` への移植時の参考素材として使う。
