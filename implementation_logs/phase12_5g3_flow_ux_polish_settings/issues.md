# Phase 12.5G-3 Issues

## 既知の課題

### 時間分割だけでは場所の意味までは完全に分からない

時刻と GPS の近接だけでフローを区切っているため、
同じ場所に長時間いた後に移動した場合でも1フローとして扱われることがある。
将来的にはアクティビティの種類（歩行・静止・移動）を使った改善が必要。

### eventMemo は PlaceGroup 単位であり写真単位ではない

1フロー内の複数写真に対してそれぞれメモを残すことはできない。
フロー全体への一言メモという位置づけに限定している。
写真単位のメモは別機能として将来検討。

### フロー分割設定は既存ノートには再推定しないと反映されない

flow-settings.tsx で設定を変えても、「この設定で再推定」ボタンを押さない限り
既存の PlaceGroup には反映されない。
UI 上で「設定変更後は再推定が必要」であることをより明確にする余地がある。

### 候補地図にピンが多い場合はまだ見づらい

MAX_SAVED_CANDIDATES = 20 の候補がすべて近距離にある場合、
280px の地図でも番号ピンが重なって見づらくなる可能性がある。
将来的にはクラスタリングやスクロール可能な候補マップを検討。

### VisitedPlacesSection ファイルは残存

VisitedPlacesSection.tsx はノート詳細から参照を外したが、
ファイル自体は削除していない。
別の画面（将来的な場所専用タブなど）で再利用する可能性を残している。
不要と確定した場合は Phase 12.5G-4 で削除を検討。
