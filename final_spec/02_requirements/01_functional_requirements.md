# Functional Requirements v2

## 1. 目的
Release v1 を、**写真を起点に地図付き思い出ノートを自動生成し、共有・共同編集・SNS共有まで実際に運用できる本番アプリ**として実装可能な粒度で定義する。  
本要件は、iOS / Android 向けに、**Firebase / Firestore / Firebase Storage / Cloud Functions 前提**で整理する。  
また、**OpenAI APIキーをモバイルアプリに直接入れない**ことを必須条件とする。

---

## 2. 要件IDルール
- F-AUTH-xxx: 認証
- F-USER-xxx: ユーザープロフィール
- F-NOTE-xxx: 思い出ノート
- F-PHOTO-xxx: 写真選択・アップロード
- F-META-xxx: メタデータ取得
- F-MAP-xxx: 地図・位置情報
- F-AI-xxx: AI生成
- F-SHARE-xxx: 共有ノート・共同編集・招待
- F-PERM-xxx: 権限管理
- F-CARD-xxx: SNS共有カード
- F-SEARCH-xxx: 検索・タグ・カレンダー
- F-SETTING-xxx: 設定・プライバシー・ヘルプ
- F-DELETE-xxx: 削除・データ削除
- F-LOG-xxx: 分析・ログ・クラッシュ
- F-NTF-xxx: 通知・招待

優先度は以下とする。
- **P0**: Release v1 必須
- **P1**: Release v1 で実装推奨
- **P2**: Release v1.1 以降推奨
